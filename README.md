**特色：**
- 🎯 实战导向，每篇文档都有可运行的代码
- 💬 语言生动，拒绝枯燥的说教
- 📝 配套练习，巩固学习成果
- 🖼️ 图文并茂，关键步骤都有截图

## 🚀 快速开始

### 环境要求
- Python 3.7+
- Windows系统（部分课程）

### 安装依赖

```bash
# 基础依赖
pip install openpyxl python-docx pymysql requests lxml selenium pywinauto pywin32 opencv-python numpy

# 或者使用requirements.txt（如果你创建了的话）
pip install -r requirements.txt
```

### 学习建议

1. **按顺序学习**：建议从第1课开始，循序渐进
2. **动手实践**：每节课的代码都要自己敲一遍
3. **完成练习**：课后练习是巩固知识的关键
4. **举一反三**：学会思路后，尝试应用到自己的项目中

## 📖 课程特色

### 🎯 你将能做出这些项目

**🔢 基础篇（学完1-9篇）**
- 猜数字小游戏 - 人机对战，猜中为止
- 九九乘法表生成器 - 双重循环经典案例
- 学生成绩管理系统 - 字典+列表综合运用
- 文件批量重命名工具 - 再也不用手动改文件名
- MySQL数据库管理工具 - 封装自己的数据库操作类

**📊 办公自动化篇（学完10-12篇）**
- Excel报表自动生成器 - 从数据库读取数据，一键生成精美报表
- 批量Word文档生成器 - 根据模板批量生成合同/证书
- Excel数据清洗工具 - 自动处理重复数据、格式转换

**🌐 网页自动化篇（学完13-19篇）**
- 自动登录脚本 - 自动填写账号密码，一键登录各种网站
- 商品价格监控器 - 定时抓取商品价格，降价自动提醒
- 新闻资讯聚合器 - 爬取多个网站新闻，整理成结构化数据
- 自动表单填写器 - 批量填写网页表单，解放双手

**🎮 游戏自动化篇（学完20-29篇）**
- 消消乐自动消除机器人 - 图像识别+算法，自动玩游戏
- 游戏挂机脚本 - 自动刷金币、自动做任务
- 游戏辅助工具 - 实时识别游戏状态，智能决策
- 带GUI的自动化控制台 - 可视化操作界面，小白也能用

### 💡 你将掌握这些技能
- **Python核心语法** - 变量、循环、函数、类，扎实基础
- **数据处理** - Excel/Word/MySQL，办公自动化必备
- **网络爬虫** - Requests、Selenium、XPath，爬取任意数据
- **图像识别** - OpenCV图像处理，计算机视觉入门
- **自动化操控** - 模拟鼠标键盘，操控任意软件
- **GUI开发** - tkinter图形界面，做出能用的工具
- **并发编程** - 多线程、多进程，提升程序效率
